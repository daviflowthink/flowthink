# Vídeo de Fundo da Homepage

## Como adicionar o vídeo de fundo

Para adicionar um vídeo de fundo à homepage, você precisa colocar um arquivo de vídeo na pasta `public/` do projeto.

### Arquivos suportados:
- `background-video.mp4` (recomendado)
- `background-video.webm` (opcional, para melhor compatibilidade)

### Especificações recomendadas do vídeo:
- **Resolução**: 1920x1080 (Full HD) ou superior
- **Duração**: 10-30 segundos (vídeos curtos funcionam melhor)
- **Formato**: MP4 com codec H.264
- **Tamanho**: Otimizar para web (máximo 10-20MB)
- **Conteúdo**: Vídeo abstrato/tecnológico que combine com o tema da FlowThink

### Como otimizar o vídeo:
1. Use ferramentas como HandBrake ou Adobe Media Encoder
2. Comprima o vídeo mantendo boa qualidade
3. Certifique-se de que o vídeo seja loopável (início e fim se conectam bem)

### Recursos gratuitos para vídeos:
- Pexels (pexels.com)
- Pixabay (pixabay.com)
- Unsplash (unsplash.com) - tem alguns vídeos
- Vídeos de tecnologia/efeitos visuais abstratos

### Funcionalidades implementadas:
- ✅ Autoplay automático
- ✅ Loop contínuo
- ✅ Muted (sem som)
- ✅ Reprodução otimizada para mobile
- ✅ Overlay escuro para melhor legibilidade do texto
- ✅ Totalmente responsivo

O vídeo será exibido em segundo plano na seção hero da homepage, atrás de todo o conteúdo mas acima dos elementos decorativos.
